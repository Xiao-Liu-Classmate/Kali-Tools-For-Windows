【简介】
Kali Tools Deployer 是一款 Kali Linux 工具 Windows 一键部署工具，
让你在 Windows 上也能快速用上 Kali Linux 的常用安全工具。


【核心功能】

两种部署模式：

1. 原生 Windows 工具包
   - 下载各工具的 Windows 原生版本
   - 一键安装/解压，自动创建快捷方式
   - 已下载的文件自动跳过，支持断点续装

2. WSL 完整 Kali 环境（推荐）
   - 一键启用 WSL 并安装 Kali Linux
   - 支持安装常用工具集或全部工具集
   - 支持 Win-KeX 图形界面
   - 兼容性最好，工具最全


【内置工具列表】

信息收集类：
  - Nmap          网络扫描与主机发现工具
  - Wireshark     网络协议分析器（抓包工具）

漏洞扫描类：
  - SQLMap        自动化 SQL 注入检测与利用工具

密码攻击类：
  - Hashcat       GPU 高速密码破解工具

Web 安全测试类：
  - Burp Suite    Web 应用安全测试平台
  - Gobuster      目录/文件/DNS 暴力枚举工具

漏洞利用类：
  - Metasploit    渗透测试框架
  - Netcat        网络瑞士军刀

无线安全类：
  - Aircrack-ng   WiFi 安全审计与密码破解工具

逆向工程类：
  - x64dbg        Windows 32/64 位调试器


【工具特点】

- 一键部署，无需手动一个个下载安装
- 完善的错误处理，下载失败自动重试
- 自动创建快捷方式，方便快速启动
- 智能跳过已下载文件，节省时间
- 两种模式自由选择，满足不同需求


【使用方法】

1. 右键 "启动部署工具.bat"，选择 "以管理员身份运行"
2. 根据菜单提示选择部署模式
3. 等待工具安装完成
4. 桌面 KaliTools 文件夹里有所有工具的快捷方式


【系统要求】

- Windows 10 / Windows 11
- 建议以管理员身份运行
- WSL 模式需要 Windows 10 1903 及以上版本


【注意事项】

- 仅供学习和授权安全测试使用，请勿用于非法用途
- 国内网络建议使用 WSL 模式，下载更稳定
- 部分工具需要 Python 等运行环境
- 安全工具可能会被杀毒软件误报，建议添加信任
- 此工具还在测试阶段，强烈建议在虚拟机中运行
